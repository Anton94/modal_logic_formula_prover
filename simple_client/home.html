Algorithms for computing satisfiability of a modal formula: an implementation
Martin Stoev, Anton Dudov and Tinko Tinchev
Sofia University

This application provides series of algorithms, used to analyze a modal formula.
  • Meassured Model
  • Simple Model
  • Connectivity Model
  • Bruteforce Model
With the first two algorithms the modal formula can be checked for satisfiability, and if it is satisfied to find its model. The connectivity algorithm is used to finds a connected model of a modal formula, if such model exists. The Bruteforce Model algorithm uses a bruteforce strategy to find a model for the given modal formula, this strategy consumes a lot of time and can be only used for a small amount of variables and operations.
